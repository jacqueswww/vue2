# Agents Guide — jacqueswww/vue2

## What This Repo Is

`jacqueswww/vue2` is a fork of [`@lastboy/vue2`](https://github.com/lastboy/vue2), which is itself a security-patched fork of the official Vue 2.7.16 source. The fork patches **CVE-2024-9506** (ReDoS vulnerability in `parseHTML`), making Vue 2 safe for production use beyond its December 2023 end-of-life.

**Package name:** `@lastboy/vue2`  
**Version:** 6.4.5  
**Package manager:** pnpm 8.9.2

## Current Task

## How to Run Tests and Build

```bash
# Install dependencies
pnpm install

# Unit tests (vitest)
pnpm run test:unit

# SSR tests
pnpm run test:ssr

# Full test suite (ts-check, types, unit, e2e, ssr, sfc)
pnpm run test

# Build all dist files
pnpm run build

# Audit
pnpm audit
```

## Cross-repo Bootstrap Vue Testing

```bash
# Clone once
git clone https://github.com/jacqueswww/bootstrap-vue.git /tmp/bootstrap-vue

# In /tmp/bootstrap-vue/package.json — set devDeps vue to local build:
# "vue": "file:/home/user/vue2"

cd /tmp/bootstrap-vue && yarn install
yarn test
```

## Project Structure

```
src/                  Vue 2 source (compiler, core, platforms, v3 compat)
packages/
  compiler-sfc/       Single File Component compiler
  server-renderer/    SSR package
  template-compiler/  Template compiler
dist/                 Built output (generated by pnpm run build)
scripts/              Build and release scripts
test/                 Unit, e2e, SSR, SFC, transition, and CVE patch tests
examples/             Standalone HTML examples + bootstrap-vue-demo/
agents/               Scratch space for agent findings and audit logs
```

## Key Files

- `package.json` — dependencies, pnpm overrides, scripts
- `pnpm-workspace.yaml` — monorepo workspace (includes `packages/*` only)
- `packages/server-renderer/package.json` — server-renderer sub-package deps
- `scripts/build.js` — main build script
- `scripts/release.js` — release automation
